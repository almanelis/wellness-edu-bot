# Wellness Telegram Bot

Телеграм-бот разработан в рамках курса "Прикладной Python" магистратуры ИИ ВШЭ.

## О проекте

Бот рассчитывает дневные нормы воды и калорий, учитывает активность и погоду,
а также ведет трекинг воды, питания и тренировок. Пользователь проходит
интерактивную настройку профиля и получает отчеты о текущем прогрессе.

## Ключевые возможности

- Настройка профиля пользователя (вес, рост, возраст, пол, активность, город).
- Расчет нормы воды с учетом активности и температуры.
- Расчет нормы калорий на основе параметров пользователя.
- Логирование воды, еды и тренировок.
- Прогресс по воде и калориям в удобном отчете.
- Получение температуры через OpenWeatherMap.
- Поиск калорийности продуктов через OpenFoodFacts.
